Lab 4: Creating an Express Web Server

This lab introduced Express and showed how to build a basic Node.js web server with routing and query parameters.

Key Skills Learned:

  • Set up a Node.js project using npm
  • Installed and used the Express library
  • Created routes and handled query parameters
  • Resolved MIME type errors in static files
  • Used Git to manage the project, including ignoring node_modules
